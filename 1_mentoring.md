    import numpy as np

    a = np.array([[-1,2,3]])

    b = np.array([[0,2,1]])

    np.dot(a,b)

`a = np.array([[-1,2,3]])`のように２重かっこにすることで、行列の掛け算になってしまう。

行列の掛け算の場合には、ここでのa,bを例にとってみると、行列aの列数と行列bの行数が一致しておく必要がある。しかし、この時、行列aの列数は３で行列bの行数は１となり、不一致となっている。そのため、次元の不一致として、`3 (dim 1) != 1 (dim 0)`というエラーが出た。

a,bの内積を計算したい場合は、行列aも行列bも１次元配列にする必要がある。
例：`a = np.array([-1,2,3])`

また、行列の乗算を計算したい場合は、公式では`numpy.matmul`を使うことが推奨されているようなので、下記ＵＲＬを参考にしてみてください。
https://numpy.org/doc/1.18/reference/generated/numpy.matmul.html#numpy.matmul
